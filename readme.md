# Рисовалка

## Описание сайта
В этой игре вы можете рисовать и использовать все возможные смайлики,размеры шрифтов,фоны,кисти, так же поиграть с исскуственным интелектом.

## Примеры запуска оффлайн версии сайта
Откройте папку AI-Painter,найдите файл index.html,кликните два раза левой кнопкой мыши по файлу и у вас в браузере октроется проект.

## Где найти сайт в интернете
Что бы найти сайт игры вам нужно - октрыть любой браузер и в строку поиска вставить или перейти по данной [Ссылке](https://gosha52-ops.github.io/AI-Painter/)

## Идеи для улучшения сайта
Нужно добавить улучшенный выбор таймера,кистей,стикеров,базовых фонов и улучшенного ии. 

## Цели проекта

Код написан в учебных целях — это урок в курсе по Python и веб-разработке онлайн шоклы [Третье Место](https://vk.com/3mesto3).
